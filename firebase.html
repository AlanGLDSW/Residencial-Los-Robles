Instrucciones para configurar Firebase Firestore

  1. Ve a la consola de Firebase
  2. Selecciona tu proyecto "residencial-los-robles"
  3. En el menú izquierdo, ve a "Firestore Database"
  4. Crea una base de datos en modo de prueba
  5. Configura las reglas de seguridad (para desarrollo puedes usar):
    rules_version = '2';
    service cloud.firestore {
      match /databases/{database}/documents {
        match /{document=**} {
          allow read, write: if true;
        }
      }
    }
  6. Tu configuración de Firebase ya está incluida en el código
